QR · CONTROL

qrctrl // configuration

部分配置保存后需重启 qrctrl 生效。
Device
手机端状态栏显示用，区分多台被控设备。
Network
默认 0.0.0.0 = 监听所有网络接口。
留空 = 从 8080 起自动找一个可用端口。
多网卡时挑哪个子网的 IP 进 QR 码。
Storage
手机上传的文件落到这里。
手机上传单文件超出会被拒绝。上限 1 TB。
Security
重启前旧 token 仍可扫码连接（如需立即作废旧 URL，保存后请重启）。
Input
1.0×
触控板拖动距离的乘数（0.1–5.0，默认 1.0）。即时生效，无需重启。
返回控制台
