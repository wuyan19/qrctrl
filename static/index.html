使用说明

连接

PC 启动后会显示二维码。用手机系统相机或微信「扫一扫」扫描，浏览器自动打开本页面。

主要功能

文本注入：手机输入的文字（含语音输入）会实时显示在 PC 当前焦点窗口。Enter 换行，点「发送」按钮提交。

双向剪贴板：拉文本 / 拉图片 把 PC 剪贴板内容取到手机；选图上传 / 粘贴截图 把手机图片写入 PC 剪贴板。

双向文件：📎 拉文件 下载 PC 剪贴板里复制的文件；📁 发送文件 把手机文件传到 PC 下载目录。

按键注入：⏎ Enter / ⇥ Tab / ⌫ Backspace 直接发送到 PC 焦点窗口。

权限

macOS 首次启动需授予「辅助功能」权限（系统设置 → 隐私与安全性）。Windows / Linux 无需特殊权限。

安全

仅本机局域网通信，每次启动随机生成 token 鉴权。关闭 PC 程序即断开所有连接。

排错

PC 的 IP 变化后需重新扫码。中文输入法无需切换，直接出字。断线会自动重连。
