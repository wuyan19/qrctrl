QR · CONTROL

qrctrl // configuration

修改后保存并重启 qrctrl 让配置生效。
Device
手机端状态栏显示用，区分多台被控设备。
Network
默认 0.0.0.0 = 监听所有网络接口。
留空 = 从 8080 起自动找一个可用端口。
多网卡时挑哪个子网的 IP 进 QR 码。
Storage
手机上传的文件落到这里。
手机上传单文件超出会被拒绝。上限 1 TB。
Security
修改 token 后需重启 qrctrl 才生效。重启前旧 token 仍可扫码连接。
Input
1.0×
触控板拖动距离的乘数（0.1–5.0，默认 1.0）。松开滑块立即生效，无需重启。
